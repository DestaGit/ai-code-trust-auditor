# AI Code Trust Auditor

A client-side web app that evaluates AI-generated code for **risk, quality, and trustworthiness**.

## 🚀 Features
- Trust Score (0–100)
- Risk detection (security, structure, reliability)
- Insight summary + top issues
- Fix guidance + execution plan
- Clean enterprise UI (sidebar + tabs)
- No backend, no APIs — runs fully in browser

## 🧠 Why This Matters
AI can generate code fast — but not always safely.  
This tool helps developers **verify before they trust**.

## 🛠 Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Client-side only (no dependencies)

## ▶️ How to Use
1. Open `index.html` in a browser  
2. Paste AI-generated code  
3. Click **Analyze**  
4. Review score, risks, and fixes  

## 📊 Metrics
- Trust Score  
- Risk count  
- Severity distribution  
- Code length  

## 📁 Project Structure
- `index.html` → Full app  
- `/docs` → Case study + demo  

## 🎯 Use Cases
- Developers validating AI code  
- Students learning code quality  
- Recruiters assessing technical thinking  

## 📌 Status
Production-ready (client-side MVP)
